============
Installation
============


Stable release
--------------

To install Queue Test, run this command in your terminal:

.. code-block:: console

    $ pip install queue_test

This is the preferred method to install Queue Test, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Queue Test can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/gvalderramos/queue_test

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://www.github.com/gvalderramos/queue_test/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .


.. _Github repo: https://www.github.com/gvalderramos/queue_test
.. _tarball: https://www.github.com/gvalderramos/queue_test/tarball/master
